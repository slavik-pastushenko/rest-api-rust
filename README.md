## Contributing

- Build an application:

```bash
cargo build
```

- Test an application:

```bash
cargo test
```

- Run an application:

```bash
cargo run
```

- Run an application in watch mode:

```bash
cargo install cargo-watch 

cargo watch -x 'run'
```

- Generate documentation in HTML format:

```bash
cargo doc --open
```
